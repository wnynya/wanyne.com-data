# network-crystal 2021-07-12
# 네트워크 크리스털 · Network Crystal
# [웹, 도구]

웹 작업을 하며 네트워크에도 관심이 생겼기에 편하게 디버그하는 도구를 만들고 싶었다.
그래서 영화에 나오는 해킹툴같은 생김새의 네트워크 디버깅 도구를 만들어 보았다.
영화에서 나오는 해킹 장면들은 터무니없는 것들이 많다. 
하지만 이 작업은 실제로 작동한다.
자바스크립트로 몇 가지 네트워크와 관련된(대부분 웹에 치중한) 디버깅 도구를 만들었고,
이를 쉽게 검색할 수 있도록 웹사이트로 만들었다.

‘마법처럼 네트워크 정보를 보여준다.’
라는 컨셉을 보이고 싶어 검색창을 수정구슬 형태로 만들었고,
이름은 네트워크 크리스털로 정했다.
