2024년 시각디자인과 소모임 연합전시 때 만든 작업입니다. 오래 된 플로피 디스켓에서 발견한 프로그램
컨셉의 체험형 작업을 하였습니다. 화면에 표시된 창에서 무작위로 선택되는 질문들에 답하면 질문 답변을 기
반으로 무작위의 그래픽을 생성해줍니다. 웹 서버와 프린터를 연결해서 생성 후 그 자리에서 바로 엽서 형태로
출력하여 가져갈 수 있게 구성하였습니다.

사용자가 프로그램을 사용하려 하면 미리 작성해 둔 질문과 예시 답변 목록 중 몇 개의 질문과 답변 세트를
선택합니다. 그 후 사용자에게 답변을 받습니다. 받은 답변을 답변에 해당하는 질문과 합쳐 단방향으로 암호화
합니다. 이 암호화된 문자들로 이미지 풀과 이미지끼리 혼합할 때 사용할 효과 풀에서 몇 가지를 선택한 후 전
부 합쳐 최종적인 그래픽을 만들어냅니다. 실제 작동하는 프로그램은 웹 페이지로 만들었기 때문에 만들어진
그래픽을 서버로 전송 후 pdf파일 형태로 바꾸어 바로 인쇄할 수 있게 만들었습니다.
